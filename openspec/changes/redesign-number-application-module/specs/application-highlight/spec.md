## MODIFIED Requirements

### Requirement: 新提交记录的高亮通过弹窗和记录视图联动实现

系统 SHALL 在用户从结果弹窗切换到记录视图时，对最近提交的记录应用视觉高亮。

#### Scenario: 从弹窗查看记录时新记录高亮
- **WHEN** 用户在结果弹窗中点击"查看记录"并切换到记录视图
- **THEN** 系统对刚刚提交的记录应用高亮样式（淡黄色背景 + 左侧蓝色边框 + 脉冲动画）

#### Scenario: 非弹窗入口进入记录视图不高亮
- **WHEN** 用户直接通过点击"记录"Tab 进入记录视图
- **THEN** 系统不应用提交高亮，仅展示正常样式（已有高亮逻辑不变，30 秒内记录自然高亮）
